## Instalación
   
   - Crear un directorio donde clonaremos la aplicación `mkdir yo_voy_por_ti`
   - Entrar al directorio `cd yo_voy_por_ti`
   - Ejecutar
     ```git clone https://gitlab.com/_yo_voy_x_ti/_yo_voy_x_ti.git .```
   - El proyecto debería haberse copiado. Revisaremos los archivos `ls`
